# StudyBuddy

Platforma za koordinaciju učenja — Pomodoro timer, flashcard sustav s
spaced repetition algoritmom, grupne study sobe s live chatom i
zajednički raspored ispita.

Timski projekt iz kolegija Napredno programiranje u programskom jeziku Java.
TVZ · Akademska godina 2025./2026.

---

## Tech stack

- Spring Boot 4 + Spring MVC REST API
- Spring Security + JWT
- JPA/Hibernate + PostgreSQL
- WebSocket + STOMP + SockJS
- Spring Mail
- Next.js
- Maven
- Vercel
- Railway
